# 📊 Data Visualization Course – Power BI & Looker Studio

## 🇫🇷 Présentation

Ce repository regroupe les supports d’un cours de data visualisation dispensé à des étudiants en informatique.

L’objectif de ce cours était d’introduire les fondamentaux de la Business Intelligence à travers la prise en main de deux outils majeurs :
- Power BI
- Looker Studio

Le parti pris pédagogique a été de travailler avec **les mêmes jeux de données sur les deux outils**, afin de permettre une compréhension concrète des concepts et une comparaison des approches.

---

## 🎯 Objectifs pédagogiques

- Comprendre les bases de la data visualisation
- Manipuler des jeux de données réels
- Construire des tableaux de bord interactifs
- Apprendre les fondamentaux du modèle de données
- Développer un regard critique sur les outils BI

---

## ⚖️ Power BI vs Looker Studio : deux approches

Bien que les deux outils poursuivent un objectif commun — produire des tableaux de bord — leur philosophie diffère :

### 🟡 Power BI
- Approche orientée **modélisation de données**
- Forte capacité de transformation et de calcul (DAX)
- Logique proche d’un modèle sémantique

### 🔵 Looker Studio
- Approche orientée **visualisation rapide**
- Connexion directe aux sources de données
- Prise en main rapide et intuitive

👉 Ce cours ne vise pas à opposer les outils, mais à proposer une **comparaison objective**, permettant aux étudiants de comprendre leurs forces et leurs limites selon les contextes.

---

## ⚖️ Power BI vs Looker Studio

Although both tools aim to create dashboards, their approaches differ:

### 🟡 Power BI
- Data modeling oriented
- Advanced calculations (DAX)
- Strong semantic layer

### 🔵 Looker Studio
- Visualization-first approach
- Direct connection to data sources
- Fast and intuitive setup

👉 This course promotes an **objective comparison**, focusing on understanding strengths and use cases rather than opposing the tools.
